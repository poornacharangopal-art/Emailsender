# 📧 Email Sender Web Application

A full-stack Email Sender web application built using **Node.js**, **HTML**, **CSS**, and **Nodemailer**. This application allows users to log in securely, send emails using their Gmail App Password, and maintain session-based authentication.

---

## 🚀 Features

* 🔐 User Login System
* 🍪 Session Authentication using Cookies
* 📧 Send Emails using Nodemailer
* 📨 Gmail App Password Authentication
* 🏠 Dashboard after Login
* 📜 Email Sending History
* 🚪 Logout Functionality
* ⚡ Lightweight and Fast

---

## 🛠️ Technologies Used

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/EmailSender.git
```

2. Navigate to the project directory

```bash
cd EmailSender
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

5. Open your browser

```text
http://localhost:3000
```

---

## 🔑 Gmail App Password Setup

1. Enable Two-Factor Authentication on your Gmail account.
2. Generate an App Password from your Google Account.
3. Use your Gmail address and App Password to log in to the application.
